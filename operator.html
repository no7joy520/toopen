算子 / 概念直觉解释（人话）常见应用
log(x) “看百分比变化”。把绝对数值变成增长率的视角。小数放大，大数压缩。
收益率对数收益率边际效应
sqrt(x) “压缩机”。大数被压缩，小数变化更明显。
波动率压缩风险调整
ts_delta(x, n) “时间序列上的差分”。就是把序列拉平，看 n 天的变化。
动量趋势强度
rank(x) “从 0 到 1 的名次”。数值大 → 排名靠前。
横截面比较标准化
group_rank(x, g) “分组里的排名”。同一行业、同一市值组里比。
行业中性化分层因子
ts_rank(x, n) “过去 n 天的相对位置”。最高是 1，最低是 0。
趋势强弱突破检测
mean(x, n) “过去 n 天的平均值”。
均线平滑噪声
cov(x, y, n) “两条线一起动吗”。协方差。
联动性相关交易
corr(x, y, n) “一起动的标准化版本”。相关系数。
成交量-价格关系对冲交易
ts_min(x, n) “过去 n 天的最低点”。
支撑位极值判断
ts_max(x, n) “过去 n 天的最高点”。
压力位极值判断
scale(x) “调成单位长度”。让总波动不变，只比形状。
归一化风险平衡
zscore(x, n) “过去 n 天离平均值有多远”。标准差单位。
异常检测均值回归
decay_linear(x, n) “最近的权重大，越久越没用”。线性衰减平均。
动量短期趋势
delay(x, n) “往前看 n 天”。历史值。
滞后因子因果检验
sign(x) “只看方向”。正是 +1，负是 -1。
趋势方向涨跌分类
prod(x, n) “n 天的连乘积”。
累计收益复利效果
abs(x) “绝对值”。只看幅度，不看方向。
波动强度幅度过滤
